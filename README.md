# Next.js Chat App

## Tech Stack

- Core: Next.js (App Router), React, TypeScript
- State: React Query, Recoil
- Styling: Emotion
- Package Manager: pnpm

## Purpose

이 프로젝트는 WebSocket 기반 채팅 기능을 직접 구현해보며,
실시간 통신과 관련된 프런트엔드 기술을 학습하기 위한 스터디용 프로젝트입니다.

채팅 도메인 자체보다는 다음과 같은 기술 요소에 집중합니다.

- Next.js(App Router) 환경에서의 클라이언트/서버 컴포넌트 구조 이해
- WebSocket을 이용한 실시간 메시지 송수신
- React Query를 활용한 메시지 캐싱 및 동기화
- Recoil을 이용한 연결 상태 및 UI 상태 관리
- Emotion을 이용한 스타일링 구성

프로젝트는 스터디 진행에 따라 점진적으로 기능을 추가하며 발전시킬 예정입니다.
